# MyBlog – Mohammad Sadman Tahiat

A modern, responsive blog website built with **HTML**, **CSS**, and **Tailwind CSS**, featuring **Dark Mode** toggle.  
The blog focuses on technology, travel, creativity, and programming.

---

## 🌟 Features

- Fully responsive layout for desktop, tablet, and mobile.
- **Dark Mode toggle** for light/dark themes.
- Grid-based article layout with smooth hover animations.
- Sidebar with author info and categories.
- Clean, professional design using **Tailwind CSS** and custom CSS.
